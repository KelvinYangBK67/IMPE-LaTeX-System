\section{Mongolian Vertical Extension}

\verb|\MOV| is tested separately because it is a vertical embedded command rather than a normal style-switch interface.

Short vertical sample:
\MOV{ᠮᠣᠩᠭᠣᠯ}

Inline sample:
Mongolian inline: \MO{ᠮᠣᠩᠭᠣᠯ} ; vertical inline: \MOV{ᠮᠣᠩᠭᠣᠯ}

Long vertical sample:
\MOV{ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ ᠦᠨ ᠦᠰᠦᠭ ᠲᠣᠪᠴᠢ ᠪᠠ ᠮᠣᠩᠭᠣᠯ ᠬᠡᠯᠡ ᠦᠨ ᠵᠢᠬᠦ ᠪᠠᠶᠢᠳᠠᠯ}
